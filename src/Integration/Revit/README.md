# ReActionAI.Integration.Revit

Модуль интеграции ReActionAI в Autodesk Revit.

## Функции

Панель ассистента (DockablePane)
Вывод сообщений модели (ChatClient)
Поддержка отправки запросов в OpenAI API
Управление конфигурацией (ConfigProvider)
Начальная структура для Pack Project

## Пути

Модуль расположен в:

src/Integration/Revit/

## Сборка

После сборки плагин автоматически копируется в:

%APPDATA%/Autodesk/Revit/Addins/2024/

### Структура в Addins:

ReActionAI.Integration.Revit.addin ReActionAI/ ReActionAI.Integration.Revit.dll ReActionAI.Abstractions.dll
