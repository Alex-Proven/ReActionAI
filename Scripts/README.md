Служебные скрипты для упаковки, сборки и публикации проекта *ReActionAI*.

## Состав
- pack_by_list.ps1 — сборка архива (PowerShell)
- pack_by_list.py — аналог на Python
- pack_by_list.sh — сборка для Linux
- Run_Packer.bat — основной пакетный сценарий запуска
- logs/ — журнал выполнения

Эти скрипты не входят в состав плагина, но используются при CI/CD и ручной сборке.
