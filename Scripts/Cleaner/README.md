# README — Cleaner Scripts (v1)

##  Назначение

Папка *Cleaner* содержит служебные скрипты, используемые для:

Лёгкой очистки рабочего репозитория ReActionAI
  (удаление bin/, obj/ и папки .vs)
Поддержания чистоты решения перед сборкой, коммитами или упаковкой Collector’ом
Автоматизации рутинной очистки через один клик

Cleaner *не участвует* в работе Revit-плагина и используется только для обслуживания репозитория.

---

##  Состав папки

*Clean-ReActionAI.bat* — основной сценарий лёгкой очистки репозитория.

---

##  Clean-ReActionAI.bat

BAT-скрипт выполняет:

Определение корня репозитория ReActionAI
   (скрипт автоматически поднимается вверх по структуре папок, пока не найдёт ReActionAI.sln).
Удаление папки *.vs* в корневом каталоге.
Рекурсивное удаление всех папок:
   - bin/
   - obj/
Вывод отчёта об удалённых каталогах.

*Важно:*
Скрипт НЕ удаляет .github, tests, .md-файлы или пользовательские директории.
Он безопасен и действует только на временные папки сборки.

---

##  Как использовать

Разместите Clean-ReActionAI.bat в папке:

##  Clean-ReActionAI.bat

Запустите двойным кликом файл Clean-ReActionAI.bat.
Дождитесь завершения очистки — скрипт покажет, что было удалено.

---

##  Примечания

Не требует прав администратора.
Работает из любой вложенной папки, так как сам ищет ReActionAI.sln.
Рекомендуется запускать перед упаковкой архива через Collector или перед отправкой коммитов.
